Conical region¶

class arepy.coord.regionCone(*args, srad=None)¶

Conical region

Parameters

  • center ([float]*3) – Center of a cone

  • radius (float) – Radius of a cone

  • theta (float) – Openning angle of a cone (radians)

Variables

  • center ([float]*3) – Center of a cone

  • radius (float) – Radius of a cone

  • theta (float) – Openning angle of a cone (radians)

Example of use:

>>> import arepy as apy
>>> import numpy as np
>>> region = apy.coord.regionCone([0.5,0.5,0.5], 0.3, np.pi/4)
>>> region.show()

Cone: center [0.5,0.5,0.5] radius 0.3 theta 0.785398
