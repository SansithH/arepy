Conical region¶

class arepy.coord.cone(*args)¶

Conical region

Parameters

  • center ([float]*3) – Center of a cone

  • radius (float) – Radius of a cone

  • theta (float) – Openning angle of a cone (radians)

Variables

  • center ([float]*3) – Center of a cone

  • radius (float) – Radius of a cone

  • theta (float) – Openning angle of a cone (radians)

Example of use:

>>> import arepy as apy
>>> import numpy as np
>>> region = apy.coord.cone([0.5,0.5,0.5], 0.3, np.pi/4)
>>> region.show()

Cone class
Center: [0.5,0.5,0.5]
Radius: 0.3
Theta: 0.785398
