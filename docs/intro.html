Introduction to the ArePy¶

Python module ArePy

ArePy is an open-source Analyzing tool for Arepo simulation data. The main module is written in Python, but some parts are also written in bash.

class arepy.showLeap(name)¶

Time measuring function

Args:

name: name of a timer

This class is mainly used to calculate the loading times of the imported modules and libraries. However, it can be used also in other parts of the code.

show(name)¶

Args:

name: name of a timer
